### Problem

假设有n个海盗成功盗取了k枚金币，在分赃的过程中产生了分歧，于是海盗们一致决定通过提议的方式来决定分赃比例，并采取如下策略：
- 通过抽签的方式决定提议顺序
- 如果有超过半数以上（包括提议人）的海盗支持提议，则提议通过
- 如果有超过半数以上的海盗反对提议，则提议不通过并且提议者将被处死
- 每个海盗必须支持或反对提议，不得弃票

我们还需作出如下假设：
- 每个海盗都将自己的生命视作最重要
- 每个海盗都是聪明的，他们能够判断在不同条件下得出自己的最优策略
- 每个海盗都是邪恶的，在获取同样金币的情况下会选择处死更多的人

### Solution

#### Case 1: n = 2 (A, B)

当只有2名海盗时，先提议的一定会被处死，后提议的海盗获得全部的金币

#### Case 2: n = 3 (A, B, C)

当有3名海盗时，如果第一个提议的海盗`A`被处死，`B`将一定会被处死，因此`B`会无条件接受`A`的提议。而由于`A`思考过后会知道`B`会无条件支持自己，`A`将会提议自己独吞所有金币。记作$[k,0,0]$。

#### Case 3: n = 4 (A, B, C, D)

当有4名海盗时，如果第一个提议的海盗`A`被处死，`B`将会提议独吞所有金币并且一定会成功，因此无论`A`提出什么提议，`B`都一定会反对。而由于`C`和`D`都是邪恶的，在不获得任何金币的情况下会选择处死更多的人，因此`A`一定不会提议自己独吞所有的金币从而使得`C`和`D`相比于`n=2`时的状况一样，所以`A`会提议让`C`和`D`获得一部分金币使他们支持自己的提议。记作$[k-2,0,1,1]$。

#### Case 4: n = 5 (A, B, C, D, E)

当有5名海盗时，如果第一个提议的海盗`A`被处死，`B`将会提议`n=4`时的提议并一定会成功，而如果`A`即使提议自己不获得任何金币，即$[0,k-2,0,1,1]$，根据海盗都是邪恶的假设，他也一定会被处死，所以他必须选择让其余4个海盗中至少2个海盗的收益大于`n=4`时的收益才能拉拢至少2个海盗。这里就需要分情况讨论：
- 第一种选择是拉拢`B`和`C`，记作$[0,k-1,1,0,0]$。
此时`B`的收益大于`n=4`时也就是自己第一个提议时的最大收益，因此`B`会同意；而`C`也比当`n=4`时即`B`第一个提议时的收益更好，`C`也会同意；`D`和`E`则不会同意。此时超过半数同意，可以执行。
- 第二种选择是拉拢`D`和`E`，记作$[k-4,0,0,2,2]$。
此时`B`一定不会同意；`C`根据假设海盗都是邪恶的，也不会同意；`D`和`E`的收益都更高了，因此一定会同意。此时超过半数同意，可以执行。

由于`A`是聪明的，因此`A`一定会选择第二种方案，记作$[k-4,0,0,2,2]$。

#### Case 4: n = 6 (A, B, C, D, E, F)

当有6名海盗时，如果第一个提议的海盗`A`被处死，`B`将会提议`n=5`时的提议并一定会成功，根据`n=5`时的理论，我们可以猜测`A`依然会选择拉拢后置位提议的海盗。如果`A`不想被处死，则必须拉拢至少3个海盗：
- 第一种选择是拉拢`B`和`C`和`D`，为了使得他们的收益更高并且`A`自己的收益尽可能高，得出提议$[1,k-3,1,1,0,0]$
- 第二种选择是拉拢`B`和`C`和`E`，此种情况和拉拢`B`和`C`和`F`的情况一样，最好的提议是$[0,k-3,1,0,3,0]$或$[0,k-3,1,0,0,3]$，而我们发现此种情况金币会不够，因此不能成为`A`所考虑的选择
- 第三种选择是拉拢`B`和`D`和`E`或`B`和`D`和`F`，这实际上同样就是第二种选择
- 第四种选择是拉拢`C`和`E`和`F`或`D`和`E`和`F`，此时最好的提议是
$[k-7,0,1,0,3,3]$或$[k-7,0,0,1,3,3]$。

由于`A`是聪明的，因此`A`一定会选择第四种方案，记作$[k-7,0,1,0,3,3]$

### General case

有如上的情况之后，我们可以发现一个规律，先提议的海盗一定有机会最大化自己的利益，因为后置位海盗受限于更坏的结果而需要被迫接受前置位海盗的提议。而最先提议的海盗一定会优先考虑拉拢后置位的海盗，因为后置位的海盗在这场博弈游戏中当人数慢慢减少时所得收益会进一步减小。

更广泛的结论是，如果我们能假设海盗是惜命的、聪明的且邪恶的，整场金币分配博弈的最终结果只与最先提议的海盗有关，并且其通过将自己被处死后潜在的前置位海盗的既得利益分配给后置位海盗的方式，使后置位海盗无法通过处死他来获得相等或更多利益，从而实现让后置位海盗支持其提议的方式使自己的利益最大化。
